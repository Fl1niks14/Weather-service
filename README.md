# Weather Service

**Weather Service** — это веб-приложение, предоставляющее актуальную информацию о погоде в различных городах. Пользователи могут быстро узнать текущие погодные условия, включая температуру, влажность, скорость ветра и другие параметры.

## 📦 Структура проекта

- `public/` — статичные файлы, включая иконки.
- `src/` — исходный код приложения.
  - `components/` — React-компоненты.
  - `services/` — сервисы для взаимодействия с внешними API.
  - `App.tsx` — основной компонент приложения.
  - `index.tsx` — точка входа приложения.
- `.gitignore` — список файлов и директорий, игнорируемых Git.
- `index.html` — основной HTML-файл.
- `package.json` — описание проекта и его зависимостей.
- `tsconfig.json` — конфигурация TypeScript.
- `vite.config.ts` — конфигурация сборщика Vite.

## ⚙️ Технологии

- **React** — библиотека для построения пользовательских интерфейсов.
- **TypeScript** — надстройка над JavaScript, добавляющая статическую типизацию.
- **Vite** — современный сборщик и сервер для разработки.
- **CSS** — стилизация компонентов.
- **API** —  Сбор информации.
